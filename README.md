# Browser Troubleshooting Toolkit

A read-only PowerShell toolkit for browser support checks.

## Features

- Browser installation detection
- Browser process context
- Internet connectivity checks
- Proxy context summary
- DNS tests for common sites
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Browser_Troubleshooting_Toolkit.ps1
```

## Safety

Diagnostic-only. It does not read browsing history, passwords, cookies, or cache.
